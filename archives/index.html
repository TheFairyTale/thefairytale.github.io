    Docker 学习小笔记
    1. 将宿主机上指定的文件夹暴露给指定容器（只能在容器被新建时使用，且要在docker desktop 设置中开启要共享的磁盘）：

      命令：
      docker run -it -v 宿主机上指定文件夹:容器内部位置 映像名
      示例，将本地C:/Users/98692/Documents 文件夹共享至容器内/media 内:
      docker run -it -v C:/Users/98692/Documents:/media tensorflow/tensorflow:latest-py3 bash
